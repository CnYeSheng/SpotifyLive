# Spotify 即時歌詞播放器 - 增強版

## 新增功能

### 1. 播放控制增強
- ✅ **隨機播放按鈕**: 開啟/關閉隨機播放模式

### 💾 雙存儲架構 (NEW!)
- **Vercel KV + localStorage**: 結合雲端和本地存儲優勢
- **即時響應**: localStorage 零延遲訪問
- **跨設備同步**: 自動同步用戶設置和歌詞
- **智能備份**: 雙重保護，永不丟失數據
- **離線支援**: 完整的離線功能
- ✅ **重複播放按鈕**: 循環切換重複播放模式（關閉 → 重複播放清單 → 單首重複）
- ✅ **智慧隨機播放**: 支援 Spotify 的智慧隨機播放功能

### 2. 播放清單功能
- ✅ **加入播放清單按鈕**: 一鍵將當前歌曲加入「已按讚的歌曲」
- ✅ **播放清單檢視**: 查看當前播放佇列
- ✅ **歌曲點擊播放**: 點擊播放清單中的歌曲直接播放

### 3. 設備投放功能
- ✅ **設備選擇按鈕**: 查看所有可用的 Spotify 設備
- ✅ **一鍵投放**: 點擊設備即可切換播放設備
- ✅ **設備狀態顯示**: 顯示當前播放設備和音量

### 4. 用戶體驗改進
- ✅ **Premium 狀態檢測**: 自動檢測用戶是否為 Premium 會員
- ✅ **功能限制提示**: 非 Premium 用戶會看到相應的功能限制提示
- ✅ **設備音量同步**: 音量控制與當前播放設備同步
- ✅ **即時狀態更新**: 播放狀態、隨機播放、重複播放狀態即時同步

### 5. 界面優化
- ✅ **新增控制按鈕**: 整齊排列的額外功能按鈕
- ✅ **模態框界面**: 美觀的播放清單和設備選擇界面
- ✅ **狀態指示**: 按鈕會根據當前狀態顯示不同的視覺效果
- ✅ **錯誤提示**: 友好的錯誤和成功提示訊息

## 使用方法

### 啟動增強版服務器
```bash
npm run enhanced
```

或者開發模式：
```bash
npm run dev-enhanced
```

### 功能說明

#### Premium 功能
以下功能需要 Spotify Premium 會員：
- 隨機播放控制
- 重複播放控制
- 設備投放
- 播放指定歌曲

#### 免費用戶功能
免費用戶仍可使用：
- 查看當前播放歌曲
- 查看歌詞
- 查看播放清單（但無法點擊播放）
- 查看可用設備（但無法切換）
- 加入已按讚的歌曲

### 按鈕說明

1. **🔀 隨機播放**: 開啟/關閉隨機播放，按鈕會在啟用時高亮顯示
2. **🔁 重複播放**: 循環切換重複模式，按鈕會根據模式顯示不同狀態
3. **➕ 加入播放清單**: 將當前歌曲加入「已按讚的歌曲」
4. **📋 播放清單**: 查看當前播放佇列
5. **📱 投放設備**: 選擇播放設備

### 設置要求

確保 `.env` 文件包含必要的 Spotify API 權限：

```env
SPOTIFY_CLIENT_ID=your_client_id
SPOTIFY_CLIENT_SECRET=your_client_secret
REDIRECT_URI=http://localhost:3000/callback
```

### API 權限範圍

增強版需要以下 Spotify API 權限：
- `user-read-currently-playing`
- `user-read-playback-state`
- `user-modify-playback-state`
- `user-read-playback-position`
- `user-library-modify`
- `user-library-read`
- `playlist-read-private`
- `playlist-read-collaborative`
- `streaming`

## 注意事項

1. **Premium 會員**: 大部分播放控制功能需要 Spotify Premium
2. **設備要求**: 需要至少一個活躍的 Spotify 設備
3. **網絡連接**: 需要穩定的網絡連接以同步播放狀態
4. **瀏覽器支援**: 建議使用現代瀏覽器以獲得最佳體驗

## 故障排除

### 常見問題

1. **按鈕顯示為禁用狀態**
   - 檢查是否為 Premium 會員
   - 確認 Spotify 應用程式正在運行

2. **設備列表為空**
   - 確保至少有一個 Spotify 設備處於活躍狀態
   - 嘗試在 Spotify 應用程式中播放音樂

3. **功能無法使用**
   - 檢查網絡連接
   - 重新授權 Spotify 帳戶
   - 確認 API 權限設置正確
