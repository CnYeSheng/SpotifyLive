## 功能測試步驟

### 1. 啟動增強版服務器
```bash
npm run start-enhanced
```

### 2. 測試音量同步
1. 在 Spotify 中調整音量
2. 檢查網頁中的音量滑塊是否同步更新
3. 拖動網頁音量滑塊，檢查是否能控制 Spotify 音量

### 3. 測試隨機播放狀態
1. 在 Spotify 中開啟隨機播放
2. 檢查網頁中的隨機播放按鈕是否高亮
3. 點擊網頁中的隨機播放按鈕，檢查狀態是否切換

### 4. 測試智慧隨機播放
1. 在 Spotify Premium 中開啟智慧隨機播放
2. 檢查按鈕是否顯示特殊樣式（綠色背景）
3. 工具提示應顯示"智慧隨機播放"

### 5. 測試重複播放
1. 點擊重複播放按鈕
2. 檢查是否循環切換：關閉 → 重複播放清單 → 單首重複
3. 按鈕應根據狀態顯示不同的工具提示

### 6. 測試播放清單功能
1. 點擊"加入播放清單"按鈕
2. 檢查是否顯示成功訊息
3. 點擊"播放清單"按鈕，檢查是否顯示當前佇列

### 7. 測試設備投放
1. 點擊"投放設備"按鈕
2. 檢查是否顯示可用設備列表
3. 點擊設備進行切換（需要 Premium）

### 8. 測試 Premium 功能限制
1. 使用免費帳戶測試
2. 檢查需要 Premium 的按鈕是否被禁用
3. 點擊禁用按鈕應顯示升級提示

### 9. 測試背景效果
1. 播放不同專輯的歌曲
2. 檢查背景是否使用專輯封面
3. 檢查是否有適當的模糊和暗化效果

### 10. 測試響應式設計
1. 在不同螢幕尺寸下測試
2. 檢查 music-card 是否適當縮放
3. 檢查按鈕是否保持可用性

## 已知限制

1. **智慧隨機播放**: 需要 Spotify API 支援，可能不是所有版本都有
2. **設備投放**: 僅限 Premium 用戶
3. **播放控制**: 某些功能需要活躍的 Spotify 設備
4. **背景圖片**: 可能受 CORS 政策影響
