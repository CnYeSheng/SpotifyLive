# 🎨 动态背景系统说明

## 现在的背景效果

您现在拥有一个**双层背景系统**：

### 📸 主背景层（专辑封面）
- **专辑封面模糊背景**：保持原有的专辑封面模糊效果
- **自动颜色提取**：从专辑封面自动提取主要颜色
- **视觉主导**：专辑封面始终是视觉焦点

### ✨ 动态效果层（叠加）
基于专辑封面颜色的动态叠加效果：
- **动态渐变**：4层基于专辑颜色的缓慢移动渐变
- **彩色粒子**：基于专辑颜色的浮动粒子系统
- **音乐波纹**：播放时的波纹扩散效果
- **光晕脉冲**：3个光晕的呼吸动画
- **节拍同步**：模拟音频分析的脉冲效果

## 🎵 效果触发时机

### 歌曲切换时：
1. 加载新专辑封面作为模糊背景
2. 自动提取封面主要颜色
3. 动态效果使用相同颜色创建叠加
4. 触发中心波纹特效

### 播放状态：
- **播放时**：动画更活跃，亮度提升
- **暂停时**：动画恢复正常速度
- **节拍时**：随机脉冲和波纹效果

1. **检查浏览器控制台**是否有JavaScript错误
2. **确认所有文件加载**（Network标签页）
3. **尝试测试页面**验证动态背景系统
4. **清除浏览器缓存**并硬刷新

- **性能优化**：使用CSS硬件加速
