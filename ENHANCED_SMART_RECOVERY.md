# 🔄 增強智能恢復功能文檔

## 概述

本文件詳細說明了 Spotify Live Lyrics 播放器中增強的智能恢復功能，該功能能夠自動處理認證問題並在會話失效時智能恢復，無需用戶手動干預。

## 🎯 主要改進

### 1. 增強的智能恢復算法
- **立即服務端刷新**：檢測到認證問題時立即嘗試服務端 token 刷新
- **漸進式重試策略**：使用指數退避算法，重試間隔為 1s, 2s, 3s, 5s
- **多端點測試**：同時測試多個 API 端點以觸發服務端刷新
- **Session 有效性檢查**：從 localStorage 恢復不同的 session ID

### 2. 自動登入觸發
- **智能檢測**：自動檢測認證失效狀態
- **延遲執行**：頁面載入後 2 秒自動執行登入檢查
- **防重複機制**：防止重複自動登入嘗試
- **用戶提示**：顯示友好的自動登入提示信息

### 3. 增強的 Token 刷新機制
- **主動刷新**：每 30 分鐘主動檢查 token 狀態
- **服務端協助**：服務端 `/api/auth-status` 端點支持主動 token 刷新
- **漸進式通知**：提供視覺反饋顯示刷新狀態
- **錯誤處理**：完善的錯誤處理和恢復機制

### 4. Session 管理優化
- **15 分鐘 Session 更新**：定期檢查和更新 session
- **動畫反饋**：提供視覺化的 session 刷新動畫
- **狀態追蹤**：詳細記錄 session 狀態變化
- **自動恢復**：session 失效時自動嘗試恢復

## ⚠️ 注意事項

1. **網絡連接**：需要穩定的網絡連接才能正常工作
2. **瀏覽器支持**：需要支持 localStorage 和現代 JavaScript 的瀏覽器
3. **Spotify 服務**：依賴 Spotify API 的可用性
4. **速率限制**：遵守 Spotify API 的速率限制規則

## 🔧 故障排除

### 常見問題
1. **恢復失敗**：檢查網絡連接和 Spotify 服務狀態
2. **自動登入失敗**：確認 Spotify 應用配置正確
3. **Token 刷新失敗**：檢查 refresh token 是否有效
4. **Session 丟失**：確認瀏覽器支持 localStorage

### 解決方案
1. 清除瀏覽器緩存和 Cookie
2. 重新授權 Spotify 應用
3. 檢查控制台日誌獲取詳細錯誤信息
4. 使用測試頁面診斷問題
